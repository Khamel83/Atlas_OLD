#  rvest: R package to scrape web data

Inspired by the Python libraries
[RoboBrowser](http://web.archive.org/web/20240618120121/http://robobrowser.readthedocs.org/en/latest/readme.html)
and
[BeautifulSoup](http://web.archive.org/web/20240618120121/http://www.crummy.com/software/BeautifulSoup/),
the [rvest package by Hadley
Wickham](http://web.archive.org/web/20240618120121/https://github.com/hadley/rvest)
helps you scrape web data via R in a similar way.

Parse tables into data frames, navigate around a website, and of course,
extract bits from a page. I’ll stick to BeautifulSoup, but I’m saving this for
later. I’m sure it’ll come in handy sooner rather than later.
