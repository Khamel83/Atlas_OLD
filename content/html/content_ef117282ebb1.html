Jim Rossman: Switching out of Windows S mode isn’t scary

S mode only allows apps from the Microsoft store and requires the default browser to be Microsoft Edge

By Jim Rossman

Technology reviewer

Jun. 16, 2024
|
Published 6:00 a.m.
|
2 min. read

Recently, I received a question from a reader, “I tried to download Scrivener, a software program for writers, and in the process, a message informed me that I could not download until I switched my Windows 11 computer out of S mode. Exiting S mode is simple. I would just go into settings and click, but a warning says this action cannot be undone, and I’m leery of anything that cannot be undone. I am wondering if there are any repercussions down the road from exiting S mode.”

Some computers ship with their copy of Windows 10 or 11 in S mode, which Microsoft says is “streamlined for security and performance.”

S mode only allows apps from the Microsoft store and requires the default browser to be Microsoft Edge and default search engine to be Microsoft Bing.

The limitation on apps only from the Microsoft app store means you can’t install software that has not been vetted by Microsoft. This is a protection measure, to be sure, but there are plenty of situations where you’d want to install other software.

S mode is kind of like training wheels for your bike. Microsoft is trying to keep you from hurting yourself, but a lot of people grow up and find a need for installing other software.

If you decide to move on from S mode, turning it off is simple, but once you enter standard mode, you cannot go back to S mode — the change is permanent.

Advertisement

After you turn off S mode, make sure you only install software from trusted sources. You’ll also want to make sure Windows Defender is turned on and up-to-date, and monitor your computer for any virus or malware activity.

To turn off S mode, open Windows settings, then open Update & Security and then Activation.

Find the Switch to Windows Home or Switch to Windows Pro section and then select the Go To Store link. You should be directed to a Switch out of S mode page. Click the Get button and then confirm the action to disable S mode permanently.
