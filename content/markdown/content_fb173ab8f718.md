<p>About one in three Americans <a href="http://web.archive.org/web/20151020063102/http://www.urban.org/UploadedPDF/413191-Delinquent-Debt-in-America.pdf">has some debt in collections</a>, according to a new report prepared by Washington think tank the Urban Institute. That&#8217;s a number that sounds really worrying, but how big a deal is it?</p>
<p>While a large debt burden is never good news, it also isn&#8217;t exactly an indicator that we&#8217;re on the precipice of another financial crisis. The report, prepared in conjunction with the Consumer Credit Research Institute based on data from credit reporting company TransUnion, says that 35% of people with credit have some debt in collections. But that number hasn&#8217;t really changed much in a decade. The report notes that a 2004 analysis of credit bureau data by the Federal Reserve showed that 36.5% had debt in collection. That&#8217;s pretty stable over a decade that also happened to include the deepest economic downturn since the Great Depression.</p>
<p>A separate report from the Federal Reserve Bank of New York also shows that <a href="http://web.archive.org/web/20151020063102/http://www.ny.frb.org/householdcredit/2014-q1/data/pdf/HHDC_2014Q1.pdf">the share of consumers with debt in collections</a> hasn&#8217;t risen considerably in recent years. The New York Fed looks at a smaller sample of the percent of people with debt held by third-party collections. That means it includes unpaid bills that are sold to collections agencies, but doesn&#8217;t count debt that is in the collections department of a big company. So an unpaid medical bill sold to a collections company is counted, while a utility bill that went to the company&#8217;s internal collections department isn&#8217;t. The Urban Institute study includes both. But the New York Fed numbers offer some sense of how the level has changed over the years.</p>

<p>The percent of people with debt held by collection agencies increased during the recession but has held pretty steady around 14% since 2010. Americans built up a lot of debt before the recession, and were hit hard by the downturn.</p>
<p>But over the past few years, things have improved markedly for debtors in the U.S. Most loan types still have more people behind on their bills than before the recession, but delinquencies are returning to normal (with the exception of student loans which still have a large share of payments past due).</p>
<p>A report this week by ratings firm Standard &amp; Poor&#8217;s noted that indebtedness in the U.S. has fallen substantially since its peak in 2007. &#8220;After the run-up of debt and the ensuing Great Recession, the household sector has gone through a long&#8211;and necessary&#8211;process of deleveraging. However, signs are now indicating that this drawn-out deleveraging might be coming to an end,&#8221; S&amp;P wrote, noting that Americans reduced their debt by a combination of borrowing less, paying down balances and defaulting on loans.</p>

<p>So, if Americans are making progress on their debt levels why hasn&#8217;t there been improvement in the percent in collections? A combination of issues is at work. For one, debt in collections stays on consumers&#8217; credit reports for at least three years. So a debt that was in default in 2007 is just coming off credit reports this year. Delinquencies built up during the dog days of the recession will only start to disappear in the next few years. Meanwhile, some people might not even know they have unpaid balances.</p>
<p>&#8220;Consumers themselves may not realize they have debt in collections. Some consumers report becoming aware of this debt only when they review their credit report,&#8221; according to the Urban Institute report.</p>
<p>Meanwhile, debt collectors have significantly grown their businesses and gotten better at finding people who owe money. &#8220;Technological innovations over the past thirty years have increased exponentially creditors&#8217; and debt collectors’ ability to obtain, store, and transfer data about consumers and their debts,&#8221; according to a 2009 Federal Trade Commission report. &#8220;In response to this new capability, many industry members use more efficient means of transferring data and store more data for longer periods of time than their predecessors.&#8221;</p>
<p>Debt is a burden to large numbers of Americans who took on more credit than they can handle. And as long as unemployment remains high and wages are stagnant, it will be difficult for consumers to pay their bills. But the large share of Americans with debt in collections has nearly as much to do with how debt is collected as how much is owed.</p>

// performs callback when element becomes visible, then removes listeners
function whenElementVisibleDo(el, cb) {
  var handler = function() {
    if (isElementInViewportAtAll(el)) {
      cb();
      removeListeners(handler);
    }
  }
  attachListeners(handler);
}

// similar to http://stackoverflow.com/a/7557433/458614 but returns true if any part of the element is in the viewport, not just the whole thing
// should work on: Chrome, Firefox, Safari, IE8+, iOS Safari, Android2+, Blackberry, Opera Mobile, IE Mobile
// doesn't handle z-index, being outside of viewable area of parent element, and possibly other edge cases
// we check if either the top OR the bottom of the element is within the viewport, while either the left right or the element is also in the viewport
function isElementInViewportAtAll (el) {
    var rect = el.getBoundingClientRect();
    var height = window.innerHeight || document.documentElement.clientHeight;
    var width = window.innerWidth || document.documentElement.clientWidth;
    return (
      (
        (rect.top <= height && rect.top >= 0) ||
        (rect.bottom <= height && rect.bottom >= 0)
      ) &&
      (
        (rect.left <= width && rect.left >= 0) ||
        (rect.right <= width && rect.right >= 0)
      )
    );
}

function attachListeners(fn) {
  if (window.addEventListener) {
    window.addEventListener('DOMContentLoaded', fn, false);
    window.addEventListener('load', fn, false);
    window.addEventListener('scroll', fn, false);
    window.addEventListener('resize', fn, false);
  } else if (window.attachEvent) { // IE fallback
    window.attachEvent('DOMContentLoaded', fn);
    window.attachEvent('load', fn);
    window.attachEvent('onscroll', fn);
    window.attachEvent('onresize', fn);
  }
}
function removeListeners(fn) {
  if (window.removeEventListener) {
    window.removeEventListener('DOMContentLoaded', fn, false);
    window.removeEventListener('load', fn, false);
    window.removeEventListener('scroll', fn, false);
    window.removeEventListener('resize', fn, false);
  } else if (window.attachEvent) { // IE fallback
    window.detachEvent('DOMContentLoaded', fn);
    window.detachEvent('load', fn);
    window.detachEvent('onscroll', fn);
    window.detachEvent('onresize', fn);
  }
}

// adds a class when top of element is off screen, removes class again when scrolling back up to that point
function addClassWhenOffTop(el, classname) {
  var scrollTopWhenAdded;
  attachListeners(function() {
    if (scrollTopWhenAdded === undefined) {
      // class has not been added , only add for ads with size less than 400
      if ( (el.getBoundingClientRect().top) < 0 && ($(el).children()[0].height < 400 ) ) {
        // it's off the top of the screen
        el.className += ' ' + classname;
        scrollTopWhenAdded = document.documentElement.scrollTop || document.body.scrollTop;
      }
    }
    else {
      if ((document.documentElement.scrollTop || document.body.scrollTop) <= scrollTopWhenAdded) {
        // we've gone back up past it
        el.className = el.className.replace(new RegExp('\\b'+classname+'\\b'), ''); // remove class. in PHP double-quoted string have to do 4 backslashes to get 2
        scrollTopWhenAdded = undefined;
      }
    }
  });
}

// okay fine we might have a duplicate declaration
function attachListeners(fn) {
  if (window.addEventListener) {
    window.addEventListener('DOMContentLoaded', fn, false);
    window.addEventListener('load', fn, false);
    window.addEventListener('scroll', fn, false);
    window.addEventListener('resize', fn, false);
  } else if (window.attachEvent) { // IE fallback
    window.attachEvent('DOMContentLoaded', fn);
    window.attachEvent('load', fn);
    window.attachEvent('onscroll', fn);
    window.attachEvent('onresize', fn);
  }
}
